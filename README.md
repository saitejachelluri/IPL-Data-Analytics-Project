# IPL Data Analytics & Business Intelligence Project 🏏📊

## Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) and Business Intelligence on IPL (Indian Premier League) datasets using Python, SQL, and Power BI. The project uncovers valuable insights related to team performance, player statistics, venue analysis, toss decisions, and season-wise trends.

---

# Objectives
- Perform data cleaning and preprocessing
- Conduct Exploratory Data Analysis (EDA)
- Analyze batting and bowling performance
- Generate SQL-based business insights
- Build an interactive Power BI dashboard
- Visualize IPL trends and patterns professionally
